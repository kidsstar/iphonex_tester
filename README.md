# iPhone X Tester

* iPhone X での見え方をテストするためのフレームを描画します

## Requirement

* Unity 2017.1
* yarn

## Install

```shell
yarn add "kidsstar/iphonex_tester#^1.0.0"
```

## Usage

* メニューの `Assets` &gt; `Install iPhone X Tester` からテスターシーンを追加してください。
